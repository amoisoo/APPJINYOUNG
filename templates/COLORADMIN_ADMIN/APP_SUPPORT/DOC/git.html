Django Ubuntu 에 Deploy

Git 파일 복사하기

서버에 파일을 다운로드한다.
git clone https://github.com/amoisoo/APPWEB2
cd APPWEB2

 

가상환경 만들기

가상환경만들기
python3 -m venv venv

가상환경을 만들기
source venv/bin/activate

가상환경을 실행한다.

pip 파일 인스톨하기

기본 장고에 필요한 라이브러리를 설치한다.
pip install -r requirements.txt

 

Django 마이그레이션
python manage.py migrate

마이그레이션
python manage.py createsupersuser

장고서버 기동하기

서버에 파일을 다운로드한다.
python manage.py runserver 192.168.219.104:80
